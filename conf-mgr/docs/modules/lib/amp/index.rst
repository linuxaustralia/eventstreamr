
:mod:`lib.amp` --- An extension of Twisted's AMP Implementaion
**************************************************************

.. module:: lib.amp

This package provides extensions for Twisted's AMP API. These include more flexible arguments and
an API to dynamically enable and disable commands.

For more information on the `AMP protocol <http://amp-protocol.net>`_ see the protocol's website; and for more information on Twisted's implementation see the documentation `provided <http://twistedmatrix.com/documents/current/core/howto/amp.html>`_ on their website.
